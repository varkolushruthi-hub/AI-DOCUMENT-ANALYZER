# 📄 AI Document Analyzer

An AI-powered document analysis application that extracts, summarizes, and answers questions from documents using Natural Language Processing (NLP) and Large Language Models (LLMs).

## 🚀 Features

- 📂 Upload PDF, DOCX, or TXT files
- 📝 Automatic document summarization
- ❓ Ask questions about the uploaded document
- 🔍 Keyword extraction
- 📊 Intelligent document insights
- ⚡ Fast and user-friendly interface
- 🤖 AI-powered semantic understanding

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript / React
- Backend: Python, Flask / FastAPI
- AI: OpenAI API / Gemini API / Hugging Face
- Document Processing: PyPDF2, pdfplumber, python-docx
- Database: SQLite / MongoDB (Optional)

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/AI-Document-Analyzer.git
```

### 2. Navigate to the project

```bash
cd AI-Document-Analyzer
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure API Key

Create a `.env` file and add your API key:

```env
API_KEY=your_api_key_here
```

### 5. Run the application

```bash
python app.py
```

The application will be available at:

```
http://localhost:5000
```
